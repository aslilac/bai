usage: okie [options] [/group...] [file...]
       okie /gleam    create files for a Gleam project
       okie /rust     create files for a Rust project
       okie /ts       create files for a TypeScript project

  -h, --help          show this help message
  -v, --version       show version information

      --set    <key>=<value> [...]   set variables persistently for use in templates
  -d, --define <key>=<value>         define a variable for use in templates

more examples:
       okie -set github.username=aslilac
       okie LICENSE -define date.year=2024
